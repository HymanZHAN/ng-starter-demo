Documentation Angular's official documentation provides you with all information you need to get started. Tooling This project is served and bundled with ESBuild. The recommended IDE setup is VSCode + Angular Language Service. If you need to test your components and web pages, check out Cypress and Cypress Component Testing.
More instructions are available in README.md. Ecosystem Get official tools and libraries for your project: Angular Services, Angular Router, Angular Testing, and Angular Dev Tools. If you need more resources, we suggest paying Awesome Angular a visit. Community Got stuck? Ask your question on Angular Community, our official Discord server, or StackOverflow. You should also follow our blog and the official @angular twitter account for latest news in the Angular world. Give us feedback! As an open source project, Angular values community feedback and contribution. You can submit issues to our Github.
